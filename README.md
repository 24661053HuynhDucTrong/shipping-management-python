# Hệ thống Quản lý Vận chuyển & Dự báo Thời gian Giao hàng

Đồ án môn Lập trình Python - Trường CĐ Kinh tế TP.HCM
Sinh viên: Huỳnh Đức Trọng | GVHD: ThS. Đinh Hoàng Gia

## Yêu cầu
- Docker Desktop đã cài sẵn
- Python 3.8+

## Hướng dẫn chạy
1. Chạy Docker SQL Server: `bash setup.sh`
2. Nạp CSDL: chạy file `instnwnd.sql` vào SQL Server
3. Chạy ứng dụng: `python main.py`
4. Mở trình duyệt: http://localhost:7860
